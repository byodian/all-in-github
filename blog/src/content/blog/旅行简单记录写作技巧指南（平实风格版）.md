> **风格关键词：** 平静、自然、真实、克制
>
> **目标：** 用简单句记录旅行，让文字有画面、有节奏，但不过度解释。

## 一、写作核心理念

> **写事实的样子，带一点人的气息。**

不需要解释，不需要抒情。\
只要写出当时发生了什么、你做了什么、感受到一点什么。

写得像你说话一样自然，\
让句子"安静地存在"，读起来顺、稳、轻。


## 二、结构框架：起 -- 行 -- 止 -- 思

| 部分         | 功能      | 内容要点      | 示例                  |
| ---------- | ------- | --------- | ------------------- |
| **起（开头）**  | 说明出发或缘由 | 时间、天气、动机  | “周末没事，就想着出去走走。”     |
| **行（过程）**  | 记录经过和观察 | 地点、动作、环境  | “路上车不多。风有点潮。”       |
| **止（收尾）**  | 结束或离开   | 动作收束、语气平稳 | “逛了一圈，中午吃了碗面，就回去了。” |
| **思（轻感受）** | 留一点感受   | 不总结、不抒情   | “不算特别好玩，但挺舒服。”      |

> 📏 节奏建议：\
> 每段 3--6 句，一气呵成；\
> 语气自然，不需要起承转合。

## 三、让句子更顺滑的技巧

### ✅ 1. 用小词柔化语气

"就""还""挺""有点"等轻语气词能让句子自然、口语化。

> ❌ 下雨了，街边全是积水，我们回酒店。\
> ✅ 下雨了，路上走不动，我们就回去了。

### ✅ 2. 减少句式重复

三句结构一样会生硬（主谓 + 主谓 + 主谓）。\
可以改变节奏，让语气有轻有重。

> ❌ 下雨了。街边全是积水。我们回酒店。\
> ✅ 下雨了，街边都是水。走了几步，就回去。

### ✅ 3. 用动作衔接逻辑

不用解释原因，用动作自然带出关系。

> ❌ 因为太热，所以进了商场。\
> ✅ 太热，就进了商场。

### ✅ 4. 用时间或环境衔接节奏

在句首加轻微的时间、环境提示，使语流顺滑。

> ✅ 天一亮，就出门。\
> ✅ 到湿地的时候，风有点潮。\
> ✅ 吃完面，太阳出来了。

### ✅ 5. 留一点空白

最后一句不要解释，让状态自己停下。

> ✅ "逛了一圈，就回去了。"\
> ✅ "不算特别好玩，但挺舒服。"

## 四、像函数一样思考句子结构

把句子想成一个最小表达单元：

``` text
sentence(scene, action, sense)
```

-   **scene（场景）**：描述当时的环境或状态\
-   **action（动作）**：记录你做了什么\
-   **sense（感受）**：轻微的感受或反应

**例：**

``` text
scene("街上人不多")
action("我们走得慢")
sense("风有点凉")

→ 输出: "街上人不多。我们走得慢。风有点凉。"
```

这三部分组合起来，就是一个自然顺口的旅行片段。

## 五、句式模板参考

> 📋 这些是最实用的「平实句式」骨架，可自由替换内容。

| 模板              | 示例               |
| --------------- | ---------------- |
| `[时间/环境]，[行为]。` | 天亮了，就出门。         |
| `[行为]，[简单反应]。`  | 吃完面，就不想动了。       |
| `[地点/人]，[动作]。`  | 岸边有人钓鱼，我们走过去看了看。 |
| `[动作1]，[动作2]。`  | 看了几眼，拍了几张。       |
| `[状态1]，[状态2]。`  | 风大，水面起了波。        |

> ✅ 原则：两句比三句顺；\
> ✅ 尽量用短句，不连太多逗号。

## 六、语言与节奏控制

| 维度     | 建议                      |
| ------ | ----------------------- |
| **语气** | 像在说话，不造句。               |
| **词汇** | 动词 > 形容词；用“走、看、吃、回、坐”等。 |
| **标点** | 想停顿 → 句号；想连贯 → 逗号。      |
| **节奏** | 一段 3–6 句，每句 10–15 字。    |
| **情绪** | 用动作体现，不用形容词解释。          |


## 七、写作练习法

1.  **去逻辑化练习**：\
    写完后删掉"因为""所以""于是"等词。
2.  **顺滑化练习**：\
    调整句序，让语气自然衔接。
3.  **函数化练习**：\
    每次出行后，写三个元素：场景、动作、感受。
4.  **留白练习**：\
    最后一行用"就"字收尾，不解释。

## 八、典型句式示例集
| 场景 | 句子示例           |
| -- | -------------- |
| 出发 | 早上出门。天阴，风不大。   |
| 行走 | 路上车不多。我们走得慢。   |
| 到达 | 到的时候太阳出来了。人不多。 |
| 休息 | 找了家小店，坐了一会儿。   |
| 吃饭 | 菜一般，但挺热乎。      |
| 回程 | 天黑了，就回去。       |
| 感受 | 没什么特别的事，但挺舒服。  |

## 九、总结

> **你要写的，不是"旅行的意义"，而是"那天的样子"。**
>
> -   不解释，只记录。
> -   不抒情，只呈现。
> -   用自然句式，让读者能看到当时的你。

写作的目标不是"文学"， 而是**让文字和那天的风一样，平常但真实。**
